• Μια φωτογραφία μπορεί να ισούται 1000 λέξεις, αλλα πλέον μπορείς να τις στέλνεις σε 4k. Χρησιμοποίησε τον νέο επιλογέα ποιότητας για να στέλνεις προαιρετικά φωτογραφίες υψηλότερης ανάλυσης. Μπορείς επίσης να ορίσεις την προκαθορισμένη αποστολή φωτογραφιών σε υψηλότερη ανάλυση στις Ρυθμίσεις > Χρήση δεδομένων.
• Οταν πατάς παρατεταμένα σε κάποιο μήνυμα, ένα νέο μενού σου επιτρέπει να αντιδράσεις, απαντήσεις ή να διαγράψεις γρήγορα.
• Παρουσιάζουμε το “My Daily Life”, ένα νέο πακέτο κινούμενων αυτοκολλήτων του καλλιτέχνη Plastic Thing. 
• Κάναμε αρκετές βελτιώσεις για να κάνουμε το σκρολάρισμα στη λίστα συνομιλιών πιο ομαλό.
